## 📝 Sobre este Repositorio

Este repositorio consiste en una plantilla lista para producción diseñada para acelerar el desarrollo de aplicaciones frontend con despliegues rápidos en **Vercel**.

> [!IMPORTANT]  
> Existen dos ramas principales: **`main`** y **`mockup-funcional`**.
>
> La rama **`main`** está destinada exclusivamente a la aplicación en **producción**.  
> La rama **`mockup-funcional`** se utiliza para el despliegue del **mockup funcional (página estática) en Vercel**.
